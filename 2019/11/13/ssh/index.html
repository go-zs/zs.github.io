CentOS配置ssh免密登录

发表于 2019-11-13

最近双十一买了新购了一台促销服务器，配置环境的时候，在配置ssh公钥免密登录的地方小小踩了一下坑…

  1. 在客户端上生成密钥对并复制

  2. 服务端配置SSH，打开密钥登录功能

  3. 服务器配置登录公钥

  4. 重启服务器SSH服务

完成之后，尝试下发现还是无法登录，检查无果。。没办法，只能查日志

恍然大悟这里还有个小坑，文件夹权限不匹配也是无法登录的~

如果尝试登录还是需要密码，可以通过服务器ssh日志错误一一排查下~
